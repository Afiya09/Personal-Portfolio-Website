First task as a MERN Stack Developer Intern at The Entrepreneurship Network — and I shipped it. 🚀

I redesigned my personal portfolio from scratch.

Not a template. Not a tutorial clone.
A full rebuild — dark editorial design, animated hero, scroll interactions, and a layout that actually tells my story.

Here's what went into it:
→ Switched from a flat cream UI to a dark palette with warm gold accents
→ Paired Playfair Display (serif) + Inter for an editorial typographic feel
→ Built a sticky nav with glassmorphism blur
→ Animated hero name reveal on load
→ Scroll-triggered fade-up animations using Intersection Observer
→ Timeline-style experience section, card-based project grid, skill chip system
→ Fully responsive — mobile down to 380px
→ Pure HTML + CSS + JS — zero frameworks, zero libraries

The brief was simple: make it eye-catching.
The goal was bigger: make it feel like *me*.

This is day one. Let's go. 💛
